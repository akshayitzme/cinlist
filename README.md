## Installation

**Method:1 Using Poetry**
```bash 
pip install poetry
```

```bash
poetry install
```
**Method: 2 Using pip**

 ```bash
 pip install -r requirements.txt
 ```

## Running
**Method: 1 Gunicorn**
```bash
gunicorn config.wsgi
```

**Method: 2 Django Development Server**
```bash
python manage.py runserver
```
